About Me

Hi, I’m Stephen Garry — a passionate and self-driven web developer currently participating in the HexCode 30 Days of JavaScript Challenge. I focus on building clean, modern, and interactive websites using HTML, CSS, and JavaScript.

This challenge has been an exciting journey of consistency and creativity. Each day, I build real-world mini projects ranging from animations and calculators to APIs and custom UI components. I'm always open to learning and pushing my limits as I work toward becoming a full-stack developer.
